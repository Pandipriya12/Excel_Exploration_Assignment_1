# Excel_Exploration_Assignment_1
The Dataset is a structured dataset with both numerical(price)and categorical data(brand,category,quality)
The datset is 35 rows 6 columns. First I used Data cleaning removing duplicate.
Now the dataset contains 32 rows and 6 columns, representing structured product data.
In this assignment, product dataset is used to perform basic statiscial operations such as SUM,COUNT,AVERAGE,MINIMUM AND MAXIMUM using excel functions.
Next using Logical function - IF PRODUCTS with price>=500 are categorized as high price
Products with price<500 are categorized as standard price
This help in grouping products based on price range.
Conditional functions - SUMIF & COUNTIF using this dataset for many products fall under lower price range (<100)
Using excel functions like LEFT,RIGHT,and MID meaningful infromation was extracted from the Product ID column.
Through data cleaning and transformation, a better foundation for accurate analysis was achieved.
